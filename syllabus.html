Welcome to the Introduction to C Programming course! Below is the course structure and syllabus for the semester.

Course Overview

This course will introduce students to the fundamental concepts of C programming. Topics include variables, control structures, functions, arrays, and pointers. By the end of the course, students will have the skills necessary to write C programs and solve computational problems.

Course Schedule

Week Topic Details
Week #1 Introduction to C and Basic I/O
  • Overview of C programming language
  • History and importance of C
  • Comparison with other programming languages
  • Real-world applications of C
  • Writing and running "Hello, World!" program
  • Understanding program structure
  • Using printf()
  • Basic variables and data types + format specifiers (int, float, char)
  • Simple input using scanf()
  • Basic arithmetic operators (+, -, *, /, %), Increment (++) and decrement (--), Compound assignment operators (+=, -=, *=, /=)
Week #2 Variables, Data Types, Operators, and Control Flow
  • In-depth variable declaration and usage: naming, #define, const
  • Comparison operators (==, !=, <, >, <=, >=)
  • Expressions and evaluation
  • Order of operations and precedence rules
  • Relational and logical operators
  • Conditional statements: if, if-else, nested if-else
  • Logical AND (&&), OR (||), NOT (!)
Week #3 Looping
  • Introduction to loops: for, while, do-while
  • Loop control statements: break, continue
  • Problem-solving techniques
  • MIDTERM EXAM PREP
Week #4 Functions and Modular Programming
  • Function fundamentals
  • What are functions?
  • Benefits of modular programming
  • Function prototypes: declaration vs definition
  • Return types, Parameters, and arguments
  • Header files and multi-file projects
  • Variable scope and static keyword
Week #5 Introduction to Arrays
  • Array fundamentals: declaring, initializing, accessing, modifying
  • Passing arrays to functions (optional: returning arrays from functions)
  • Multidimensional arrays: 2D arrays, matrix operations
  • Character arrays (C-style strings): initialization, basic string operations
  • MIDTERM
Week #6 Strings
  • Common string functions: strlen(), strcpy()
  • String manipulation exercises
  • Name formatting, simple text processing
  • Review of previous weeks
Week #7 Pointers + Dynamic Memory Allocation
  • Introduction to Pointers
  • Pointer variables (int *ptr), Pointer arithmetic (ptr++, ptr--)
  • Pointers and arrays
  • Passing pointers to functions
  • Dynamic Memory Allocation
Week #8 Structs + Review of Dynamic Memory Allocation
  • Review of Week #6 and #7
  • Structures in C
Week #9 Simple Preprocessor + Review of Concepts
  • Basic preprocessor directives (#include)
  • Simple macro definitions
  • Conditional compilation basics
  • Simplified #ifdef usage
  • Introduction to basic error handling
  • Review and exam preparation activities
Week #10 Review + Final Exam
  • Review of key concepts
  • Practice problems
  • Project due (last day of class)
  • Final Exam
